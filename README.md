# 🍽️ A Smart Chatbot System for Minimizing Food Wastage Through Ingredient-Based Recipe Suggestions

## 📌 Project Overview

This project is a research-based Smart Chatbot System designed to minimize food wastage by providing intelligent recipe suggestions based on available ingredients.

The system allows users to input ingredients and receive personalized recipe recommendations through a chatbot interface powered by Natural Language Processing (NLP) and Machine Learning techniques.

The application is developed using Flutter for the mobile frontend, FastAPI for backend services, MySQL for database management, and Firebase Authentication for secure login (Google & Email authentication).

---

## 🎯 Research Problem

Food wastage is a global issue caused by poor meal planning and lack of knowledge about recipes that can be prepared using available ingredients.

Many users:
- Do not know what to cook with available ingredients
- Waste usable food items
- Lack personalized cooking assistance
- Struggle with meal planning

This project solves these issues using an intelligent recipe recommendation chatbot.

## 📊 Machine Learning Approach

### TF-IDF (Term Frequency - Inverse Document Frequency)
Used to convert ingredient text into numerical vectors for similarity matching.

### Logistic Regression
Used for classification and predicting the most relevant recipe based on input ingredients.
